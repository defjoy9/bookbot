# BookBot

A simple text analysis tool that counts words and character frequency in books.

## Usage
```bash
python main.py <path_to_book>
```

## Example
```bash
python main.py books/frankenstein.txt
```

## Output
- Total word count
- Frequency of each alphabetic character
- Sorted character counts

## Example Output
